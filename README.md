# 日本語 Complete Japanese Cheat Sheet

A comprehensive, searchable Japanese language reference guide with Bengali translations. This single-page web application provides quick access to essential Japanese grammar, vocabulary, and writing systems.

## Features

- **📱 Responsive Design** - Fully optimized for mobile, tablet, and desktop
- **🔍 Live Search** - Real-time filtering across all content
- **🎯 Quick Navigation** - Sticky header with tabbed navigation for easy access
- **🌏 Multi-language Support** - Japanese, Romaji, English, and Bengali translations
- **✨ Clean UI** - Elegant design with smooth animations and transitions

## Content Sections

### Writing Systems

- **Hiragana** - Complete character chart with romaji and Bengali
- **Katakana** - Complete character chart with romaji and Bengali
- **Kanji** - Common kanji with readings and meanings

### Grammar & Particles

- **Particles** - Detailed explanations with examples (は, が, を, に, で, etc.)
- **Verbs** - Conjugation tables and usage patterns
- **Adjectives** - い-adjectives and な-adjectives with conjugations
- **Advanced Grammar** - Complex grammar patterns and structures

### Vocabulary

- **Numbers** - Counting systems and number expressions
- **Time** - Time-related vocabulary and expressions
- **Kinship Terms** - Family relationship vocabulary
- **Vocabulary** - Common words and phrases organized by category

### Cultural & Advanced

- **Keigo (敬語)** - Polite and honorific language
- **Natural Japanese** - Colloquial expressions and phrases
- **Spoken Japanese** - Conversational patterns
- **Expressions** - Common phrases and idioms
- **四字熟語 (Yojijukugo)** - Four-character idioms

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

### Search Functionality

Type in the search box to filter content across all sections. The search works with:

- Japanese characters (hiragana, katakana, kanji)
- Romaji
- English translations
- Bengali translations

### Navigation

- Click on any tab in the navigation bar to jump to that section
- The navigation bar stays visible as you scroll
- All sections are accessible via anchor links

## Technical Details

- **Single HTML file** - Entire application in one file for easy deployment

- **Google Fonts** - Noto Sans JP, Noto Sans Bengali, Libre Baskerville, JetBrains Mono
